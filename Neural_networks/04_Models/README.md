# Language Models Overview

This folder contains information on various language models:

1. BERT and its variants
2. GPT models (GPT-2, GPT-3, etc.)
3. T5 (Text-to-Text Transfer Transformer)
4. XLNet
5. RoBERTa
6. ALBERT
7. DistilBERT
8. ELECTRA
9. BART
10. Transformer-XL

Each model has its own dedicated file with detailed explanations, architecture, use cases, and examples.
